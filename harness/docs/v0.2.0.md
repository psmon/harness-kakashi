# v0.2.0 - init 기능 및 모드 시스템 보강

## 변경사항

- **Mode D: Initialize** — `/harness-kakashi-creator init` 명령으로 하네스 초기화
  - templates/ 디렉토리에 초기화 템플릿 내장
  - 사용자 입력(이름, 설명) 기반 harness.config.json 자동 생성
  - 기존 harness/ 존재 시 덮어쓰기 확인
- **모드 판별 시스템** — 개선부(Mode A~D) / 수행부 분리
  - Mode A: Log & Eval (조련사 트리거 매칭)
  - Mode B: Suggestion Tip (새 에이전트/워크플로우 제안)
  - Mode C: Kakashi Copy (스킬 복사, skill-creator 위임)
  - Mode D: Initialize (하네스 초기화)
- **references/** 디렉토리 추가
  - layer-guide.md — 3-Layer 아키텍처 상세 가이드
  - evaluation.md — 조련사 3축 평가 체계 (A~D / 1~5 / L1~L5)
  - tamer-agent.md — 조련사 에이전트 상세 정의
- **에이전트 frontmatter** — triggers, description 필드 추가
- **로그 체계** — 경로 패턴, 형식, 필수 기록 규칙 정의
- **피드백 루프** — 평가 → 개선 제안 → 승인 → 적용 → 재평가 사이클

## 참고

AgentWin 프로젝트의 harness-creator-kakashi 최신 기능을 분석하여 보강.
