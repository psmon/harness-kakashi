# v1.3.0 — 차크라 카카시 영입 (토큰 효율 감사 스킬)

**날짜**: 2026-04-20

## 변경 사항

### 신규 스킬 (1종) — 플러그인 하위 컴포넌트

다른 브랜치(`AgentWin/.claude/skills/harness-chakra-kakashi`)에서 개발 중이던
**차크라 카카시(Chakra Kakashi)** 스킬을 카카시 하네스 플러그인의 **독립 하위 컴포넌트**로 영입.

| 스킬 | 역할 | 위치 |
|------|------|------|
| harness-chakra-kakashi | 토큰(차크라) 사용량 3인칭 사후 감사 | `plugins/harness-kakashi/skills/harness-chakra-kakashi/` |

**핵심 기능**:
- 클로드코드 2.0 세션의 토큰 경제성을 5축으로 평가 (차크라 총량 / 순환 효율 / 술법 정밀도 / 인술 선택 / 분신술 절제)
- JSONL 트랜스크립트 + `/cost` + `/context` + `ccusage` CLI 통합 감사
- 마스킹 정책으로 원문/경로/비밀 비기록
- 하네스 있으면 `harness/agents/chakra-auditor.md` 연동, 없으면 단독 동작

**영입 시 어댑테이션**:
- 원본의 자동 발동 트리거 중 `agent-zero`, `agent-zero-vibe` 참조 제거 (AgentWin 전용 스킬이라 본 플러그인 사용자에게 부재)
- 카카시 하네스 수행부 엔진 종료 트리거로 일반화
- 본 플러그인의 독립 하위 컴포넌트임을 명시 (`harness-kakashi-creator`와 독립 동작)

**참조 문서 3종 (verbatim)**:
- `references/jutsu-patterns.md` — 효율 술법(DO) / 차크라 낭비 안티패턴(DON'T)
- `references/chakra-masking.md` — 로그 마스킹 정책 (개인정보 보호)
- `references/claude-code-2-token-sources.md` — 토큰 관측 소스 레퍼런스

### 플러그인 구조 변경

`plugins/harness-kakashi/skills/` 가 단일 스킬에서 **2-스킬 구조**로 확장.

```
plugins/harness-kakashi/
├── .claude-plugin/plugin.json
└── skills/
    ├── harness-kakashi-creator/   ← 기존: 하네스 통합 관리
    └── harness-chakra-kakashi/    ← 신규: 토큰 감사 (독립)
```

두 스킬은 서로 독립이며, 사용자는 필요에 따라 어느 쪽이든 단독 호출 가능.

### 버전 정렬 (드리프트 정리)

이전까지 두 트랙이 드리프트 상태였음:
- `harness/harness.config.json` → 1.2.0 (내부 진화 추적)
- `.claude-plugin/marketplace.json` + `plugin.json` → 0.1.0 (초기값 그대로)

본 릴리스에서 **세 파일을 1.3.0으로 정렬**. 이후로는 harness-build 규칙대로 동기화 유지.

### harness.config.json 보강

`plugins` 필드 추가 — 본 저장소에서 배포되는 플러그인과 그 하위 스킬 목록을 명시:

```json
"plugins": {
  "harness-kakashi": {
    "skills": ["harness-kakashi-creator", "harness-chakra-kakashi"]
  }
}
```

이 필드는 빌드 상태 점검 시 plugins/ 디렉토리 실제 구조와 대조하는 용도.

## 사용자 영향

- 기존 사용자: `harness-kakashi-creator` 동작은 그대로. 차크라 스킬은 호출 시에만 발동.
- 신규 사용자: 플러그인 설치 시 두 스킬 모두 사용 가능. 각자 독립 활성화.
- 자동 트리거 없음 — 수행부 엔진이 명시적으로 위임할 때만 차크라가 깨어남.

## 버전

- 이전: 1.2.0 (.NET 전문가 6명 영입)
- 현재: 1.3.0 (차크라 카카시 영입 + 플러그인/하네스 버전 정렬)
- 총 플러그인 스킬: **2종** (harness-kakashi-creator, harness-chakra-kakashi)
- 총 에이전트: 12명 (변동 없음)

## 참고 모드 vs 플러그인 모드

본 영입은 **플러그인 모드** 작업이다.
- 원본 차크라 스킬은 AgentWin 저장소에서 참고 모드로 개발됨 (플러그인 배포 무관)
- 본 저장소(harness-kakashi)는 **Claude 플러그인 배포가 핵심**이므로
  `plugins/harness-kakashi/skills/` 가 source of truth
- `.claude/skills/`는 로컬 테스트용 미러 (개발 시 동기화 필수)
