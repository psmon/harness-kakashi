### 신규 지식 문서

- `harness/knowledge/aaronontheweb-dotnet-skills-index.md` — 업스트림 30 skills + 6 agents 인덱스, 라우팅 힌트
- `harness/knowledge/dotnet-expert-evaluation.md` — 5대 관문(G1–G5) 기반 통합 평가 체계

### 평가 체계

6인 전문가 기준 **5대 관문(Gate)** 도입:
- G1 액터/분산 건전성 (akka-net-specialist)
- G2 동시성 안전성 (dotnet-concurrency-specialist)
- G3 성능/회귀 (benchmark-designer + performance-analyst)
- G4 소스 생성기 정합성 (roslyn-incremental-generator-specialist)
- G5 문서 품질 (docfx-specialist)

프로젝트 성격에 따라 관문을 **선택 적용**한다:
- 라이브러리 프로젝트: G3 + G4 + G5
- 서비스 프로젝트: G1 + G2 + G3
- 문서 저장소: G5 only
